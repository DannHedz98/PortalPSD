BMC ITSM - Remedy en la Nube - Manual del usuario

El contenido de estas páginas muestran la forma de operación relacionada a los procesos que se llevan a cabo en América Móvil.

Si usted es un usuario nuevo, inicie con el menú de la izquierda seleccionando el proceso del cual requiere información. O siga las siguientes ligas acerca del modelo estándar y sus procesos.

  • Modelo Estándar
  • Proceso de Solicitud de Accesos
  • Proceso Correctivo
  • Proceso de Emergencias
  • Proceso Estadístico
  • Proceso de Intervenciones en la Red
  • Proceso Preventivo
  • Proceso Recibir e Integrar Infraestructura de la Red

Si usted es un usuario que ya conoce Remedy ITSM, seleccione del menú superior el tema acerca del cual requiere información. O siga estas ligas acerca del módulo del que desea información.

  • Generales - Conozca Remedy ITSM
  • Gestión de Incidentes
  • Gestión de Cambios
  • Gestión de órdenes de trabajo
  • Gestión de Conocimientos
  • Tareas
  • Gestión de Problemas
